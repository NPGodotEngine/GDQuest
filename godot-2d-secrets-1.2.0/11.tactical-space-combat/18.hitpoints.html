Weapons and Hitpoints

In this chapter, we’ll add winning and losing conditions by implementing ship hitpoints.

We’ll see how to handle weapons firing and registering damage to the ships. We have to transfer weapon parameters across viewports for both projectiles and lasers. Since we already implemented most of the building blocks, we can see how easy it is to expand our previous work.

We’ll also add an attack label feedback animation when weapons interact with the ship rooms.

Check out this video to get an idea of how we’ll progress by the end of this lesson.

Handling weapon attacks

To affect ships’ hitpoints, we need to assign an attack value to each weapon. We start with updating the weapon scripts and then moving on to handling the interaction.

Transfering weapon data over viewports

Open the Weapon.gd script and add a new attack variable at the top:

I limited the value range to (0, 5) and set the default value to 2 as a middle ground.

Handling the attacks

We start by setting the Room -> HitArea2D node to the proper physics layer to interact with the weapons from the other ship.

Open the Room.gd script and add a reference variable to the HitArea2D node:

Then update its collision_mask property in the setup() function to the proper values depending on whether it’s the player ship or the AI ship.

We’re now ready to handle the damage taken in the ShipTemplate.gd script. Open it then lets start by adding a signal at the top:

signal hitpoints_changed(hitpoints, is_player)

We use this signal to update the UI Hitpoints labels. Since we have one script for both player and AI, we distinguish between them with the second parameter: is_player.

Next, let’s preload the AttackLabel and add a hitpoints variable for the ships’ health.

# ...
# const LaserTracker := preload("Weapons/LaserTracker.tscn")
const AttackLabel := preload("Weapons/AttackLabel.tscn")

export (int, 0, 30) var hitpoints := 30

We now move on to handling the attacks via projectiles and lasers:

## Handles projectiles.
func _on_RoomHitArea2D_body_entered(body: RigidBody2D, room: Room) -> void:
    # We make sure that the projectiles interact with the correct `Room` by
    # checking their positions.
    if not room.position.is_equal_approx(body.params.target_position):
        return

    body.queue_free()
    _take_damage(body.params.attack, room.position)


## Handles lasers.
func _on_RoomArea2D_area_entered(area: Area2D, room: Room) -> void:
    if area.is_in_group("laser"):
        _take_damage(area.params.attack, room.position)

We need to implement a new function called _take_damage() that handles decreasing the hitpoints value and adding attack labels for player feedback:

func _take_damage(attack: int, object_position: Vector2) -> void:
    var attack_label := AttackLabel.instance()
    attack_label.setup(attack, object_position)
    add_child(attack_label)

    hitpoints -= attack
    hitpoints = max(0, hitpoints)
    emit_signal("hitpoints_changed", hitpoints, is_in_group("player"))

To finish off updating the ShipTemplate.gd script, we need to connect the proper signals to the _on_Room*() functions. We do this in _ready_not_editor_hint():

func _ready_not_editor_hint() -> void:
    # ...
    # for room in rooms.get_children():
        room.connect("area_entered", self, "_on_RoomArea2D_area_entered", [room])
        room.hit_area.connect("body_entered", self, "_on_RoomHitArea2D_body_entered", [room])

Note that we send the extra room parameter to each function. That’s for adding the AttackLabel in proper positions when the weapons interact with specific rooms.

Win conditions

The only thing left for us to do is update the UI HitPoints labels whenever the hitpoints_changed signal emits and switch the scene when one of the ships hits 0 hitpoints.

We do this in the main TacticalSpaceCombat.gd script. Open the file in the script editor, and let’s start by adding the path to the End scene so we can switch to it:

# ...
# const UIWeapon = preload("TacticalSpaceCombat/UI/UIWeapon.tscn")

const END_SCENE = "TacticalSpaceCombat/End.tscn"

We also need references to the new labels:

# ...
# onready var ui_systems: HBoxContainer = $UI/Systems
onready var ui_hitpoints_player: Label = $UI/HitPoints
onready var ui_hitpoints_ai: Label = $ViewportContainer/Viewport/UI/HitPoints

We connect the ships’ hitpoints_changed signal in the _ready() function:

func _ready() -> void:
    ship_player.connect("hitpoints_changed", self, "_on_Ship_hitpoints_changed")
    ship_ai.connect("hitpoints_changed", self, "_on_Ship_hitpoints_changed")
    # ui_doors.connect("pressed", ship_player, "_on_UIDoorsButton_pressed")
    # ...
    # _ready_weapons_player()

    # We finish off by manually calling the function to overwrite the default
    # `Label` text.
    _on_Ship_hitpoints_changed(ship_player.hitpoints, true)
    _on_Ship_hitpoints_changed(ship_ai.hitpoints, false)

Finally, we add the missing function to the script:

func _on_Ship_hitpoints_changed(hitpoints: int, is_player: bool) -> void:
    var label := ui_hitpoints_player if is_player else ui_hitpoints_ai
    label.text = "HP: %d" % hitpoints
    if hitpoints == 0:
        Global.winner_is_player = not is_player
        get_tree().change_scene(END_SCENE)

Conclusion

In this lesson, we hit a new milestone by handling weapon attacks. We now have win-or-lose states.

In implementing these, we saw how to transfer information from one viewport to another via signals when firing the weapons.

We also saw how to handle updating our UI when the ship’s hitpoints change when attacked.

We also saw how to use the AnimationPlayer for more than pure animations efficiently. We made a self-contained attack label that cleans itself up by calling queue_free() at the end of the animation.

We saw how to align labels, so they grow evenly to keep things neat and centered.

Finally, we saw how to persist data across scenes using our custom Global singleton to update the winner information.

res://TacticalSpaceCombat/Ship/Weapons/WeaponProjectile.gd:

tool
extends Weapon

signal fired
signal projectile_exited(params)

const Projectile := preload("Projectile.tscn")

var target_position := Vector2.INF

var _physics_layer := -1


func setup(physics_layer: int) -> void:
    _physics_layer = physics_layer


func _get_configuration_warning() -> String:
    var parent := get_parent()
    var is_verified := parent != null and parent is ControllerAIProjectile or parent is ControllerPlayerProjectile
    return "" if is_verified else "WeaponProjectile needs to be a parent of Controller*Projectile"


func fire() -> void:
    if not can_fire():
        return

    self.is_charging = true
    var projectile: RigidBody2D = Projectile.instance()
    projectile.linear_velocity = projectile.linear_velocity.rotated(global_rotation)
    var params := {
        "physics_layer": _physics_layer,
        "target_position": target_position,
        "attack": attack
    }
    projectile.connect("tree_exited", self, "emit_signal", ["projectile_exited", params])
    add_child(projectile)
    emit_signal("fired")


func can_fire() -> bool:
    return not is_charging and target_position != Vector2.INF

    _rng.randomize()


func _on_Weapon_projectile_exited(params: Dictionary) -> void:
    var projectile: RigidBody2D = Projectile.instance()
    var spawn_position := _mean_position + Utils.randvf_circle(_rng, projectile.MAX_DISTANCE)
    var direction: Vector2 = (params.target_position - spawn_position).normalized()
    projectile.collision_layer = params.physics_layer
    projectile.position = spawn_position
    projectile.linear_velocity = direction * projectile.linear_velocity.length()
    projectile.rotation = direction.angle()
    projectile.max_distance *= 2
    projectile.params = params
    add_child(projectile)

res://TacticalSpaceCombat/Ship/Weapons/LaserTracker.gd:

extends Node2D

signal targeted(msg)

const LINE_DEFAULT := PoolVector2Array([Vector2.INF, Vector2.INF])

var _is_targeting := false
var _targeting_length := 0
var _rng := RandomNumberGenerator.new()
var _rooms: Node2D = null

onready var tween: Tween = $Tween
onready var area: Area2D = $Area2D
onready var line: Line2D = $Line2D
onready var target_line: Line2D = $TargetLine2D


func setup(color: Color, rooms: Node2D) -> void:
    _rooms = rooms
    line.default_color = color
    target_line.default_color = color


func _ready() -> void:
    _rng.randomize()
    target_line.points = LINE_DEFAULT
    line.points = LINE_DEFAULT


func _unhandled_input(event: InputEvent) -> void:
    if not (event is InputEventMouse and _is_targeting):
        return

    if event.is_action_pressed("left_click"):
        target_line.points[0] = get_local_mouse_position()
    elif target_line.points[0] != Vector2.INF and event is InputEventMouseMotion:
        var offset: Vector2 = get_local_mouse_position() - target_line.points[0]
        offset = offset.clamped(_targeting_length)
        target_line.points[1] = target_line.points[0] + offset
    elif event.is_action_released("left_click") and target_line.points[1] != Vector2.INF:
        _is_targeting = false
        emit_signal("targeted", {"type": Controller.Type.LASER})


func _on_Controller_targeting(msg: Dictionary) -> void:
    match msg:
        {"targeting_length": var targeting_length, "is_targeting": var is_targeting}:
            _is_targeting = is_targeting
            _targeting_length = targeting_length
            if _is_targeting:
                target_line.points = LINE_DEFAULT
        {"targeting_length": var targeting_length}:
            target_line.points = _rooms.get_laser_points(targeting_length)
            emit_signal("targeted", {"type": Controller.Type.LASER})


func _on_Weapon_fire_started(params: Dictionary) -> void:
    if Vector2.INF in target_line.points:
        return

    area.set_deferred("monitorable", true)
    area.params = params

    line.points[0] = _rooms.mean_position + Utils.randvf_circle(_rng, Projectile.MAX_DISTANCE)
    tween.interpolate_method(
        self, "_swipe_laser", target_line.points[0], target_line.points[1], params.duration
    )
    tween.start()


func _on_Weapon_fire_stopped() -> void:
    tween.remove_all()
    target_line.points = LINE_DEFAULT
    line.points = LINE_DEFAULT
    area.position = Vector2.ZERO
    area.set_deferred("monitorable", false)


func _swipe_laser(offset: Vector2) -> void:
    line.points[1] = offset
    area.position = line.points[1]

res://TacticalSpaceCombat/Ship/Rooms/Room.gd:

tool
class_name Room
extends Area2D

signal targeted(msg)

export var size := Vector2.ONE setget set_size

var _target_index := -1

var _rng := RandomNumberGenerator.new()
var _entrances := {}
var _area := 0
var _top_left := Vector2.ZERO
var _bottom_right := Vector2.ZERO

var _iter_index := 0

var _tilemap: TileMap = null

onready var collision_shape: CollisionShape2D = $CollisionShape2D
onready var feedback: NinePatchRect = $Feedback
onready var sprite_target: Sprite = $SpriteTarget
onready var hit_area: Area2D = $HitArea2D


func setup(tilemap: TileMap) -> void:
    _tilemap = tilemap
    _setup_extents()

    if not Engine.editor_hint:
        hit_area.collision_mask = (
            Global.Layers.SHIPPLAYER
            if owner.is_in_group("player")
            else Global.Layers.SHIPAI
        )

    _area = size.x * size.y
    _top_left = _tilemap.world_to_map(position - collision_shape.shape.extents)
    _bottom_right = _top_left + size

    feedback.rect_position -= collision_shape.shape.extents
    feedback.rect_size = 2 * collision_shape.shape.extents


func _setup_extents() -> void:
    if _tilemap != null:
        collision_shape.shape.extents = 0.5 * _tilemap.map_to_world(size)


func _ready() -> void:
    connect("mouse_entered", self, "_on_mouse_entered_exited", [true])
    connect("mouse_exited", self, "_on_mouse_entered_exited", [false])
    connect("area_entered", self, "_on_area_entered")
    connect("input_event", self, "_on_input_event")
    _rng.randomize()


func _on_input_event(_v: Viewport, event: InputEvent, _s: int) -> void:
    if (
        event.is_action_pressed("left_click")
        and Input.get_current_cursor_shape() == Input.CURSOR_CROSS
        and _target_index != -1
    ):
        sprite_target.visible = true
        sprite_target.get_child(_target_index).visible = true
        emit_signal(
            "targeted",
            {
                "type": Controller.Type.PROJECTILE,
                "index": _target_index,
                "target_position": position
            }
        )
        _target_index = -1


func _on_mouse_entered_exited(has_entered: bool) -> void:
    feedback.visible = has_entered
    var group := "selected-room"
    if has_entered:
        add_to_group(group)
    elif is_in_group(group):
        remove_from_group(group)


func _on_area_entered(area: Area2D) -> void:
    if area.is_in_group("door"):
        var entrance := position - area.position
        entrance = entrance.project(Vector2.DOWN.rotated(area.rotation)).normalized()
        entrance *= 0.5 * _tilemap.cell_size.x
        entrance += area.position
        entrance = _tilemap.world_to_map(entrance)
        _entrances[entrance] = null


func _on_Controller_targeting(msg: Dictionary) -> void:
