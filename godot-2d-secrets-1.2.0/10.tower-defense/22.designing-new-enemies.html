Designing New Enemies

Following our content creation section, let’s see how we can design new enemies to use in harder waves.

Open the res://Actors/Enemies/Enemy.tscn scene and create a new inherited scene using the enemy as base scene. This is going to be a tank enemy, so save this inherited scene as res://Actors/Enemies/ToughEnemy.tscn.

We change the appearance of this new enemy by toggling on the CutoutCharacter Editable Children and using the TextureRegion editor to change sprites one to four to the red-ish circle instead of the orange one. Sprite5 shows the eyes so we don’t need to change it.

You might think we’ve been using this feature recklessly, but note that we didn’t change anything in the structure, architecture or engineering of these nodes. For instance, it would be a HUGE mistake if we changed the CollisionShape2D’s radius instead of setting the weapon’s fire range using the Fire Range property. So, be mindful about how you edit children in instance scenes. With great power comes great responsibility.

We can also scale up the CutoutCharacter to give the visual cue that this enemy is tougher than the orange one. Change its scale to 1.2 in both axes. As the enemy looks bigger, towers should detect it easier. In VisibleArea2D > CollisionShape2D, create a new Circle2D and make its radius cover the ToughEnemy.

To actually make it tougher and improve its resistance to damage, increase its armor and health. In the HurtBoxArea2D increase the Armor property to 2. Then select the root node and increase its Max Health to 12.

Since it looks heavier than the normal enemy, we make it move slower. Set its speed to 128.0. Because its a harder enemy to defeat, we can make it drop more gold. I think 75 sounds fair enough.

There we have it, a brand new enemy to spawn. From here you can experiment with other enemy types. Another could be quicker and smaller, but would have less health and no armor. Be creative!

How can we use these new enemies in our game?

Remember in the level lesson we designed two different waves of enemies? We can do the same thing. Create a new Wave scene and add some enemies as children. Here you can design the wave as you wish. Be aware that the order which the enemies appear in the hierarchy affects how the appear in the game. You can tweak the Wave > Enemy Time Interval to make enemies spawn closer to each other.
